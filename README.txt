About:

EuSDLGfx2 is a wrapper of SDL2_Gfx. It is a set of graphics instructions you can use to extend SDL.
You can use it to draw lines, boxes, strings, etc. It is meant to extend the capabilities of the
SDL multimedia library. This wrapper was written in Euphoria 4.1.0 Beta 2. This wrapper is meant
for use with EuSDL2. You will EuSDL2 in order to use this wrapper as it is an extension of the SDL
library.
